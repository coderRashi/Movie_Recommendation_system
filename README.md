# Movie_Recommendation_system
A machine learning project that recommends movies to users based on their preferences using content-based filtering, collaborative filtering, and hybrid approaches. Built with Python, scikit-learn, and pandas. Includes interactive notebooks, a clean dataset, and evaluation metrics like cosine similarity and RMSE.
A content-based recommendation system suggests movies by analyzing the attributes of films and matching them with a user’s past preferences.

How it works?

Each movie is represented by features such as genre, director, cast, keywords, or even text descriptions.

The system builds a profile of the user’s interests based on the movies they’ve liked or rated highly.

It then calculates similarity (often using cosine similarity) between the user’s preferred movies and other movies in the dataset.

Movies with the highest similarity scores are recommended.
